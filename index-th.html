บริการ

แพลตฟอร์มยืนยันตัวตนของเรารองรับวงจรการปฏิบัติตามกฎหมายแบบสมบูรณ์ ตั้งแต่การคัดกรองเบื้องต้นและการเพิ่มผู้ใช้ไปจนถึงการตรวจสอบอย่างต่อเนื่องและการป้องกันการฉ้อโกง ด้วยความเชี่ยวชาญเฉพาะภูมิภาค ที่ขับเคลื่อนด้วยโครงสร้างพื้นฐานระบุตัวตนที่ได้รับการยอมรับในระดับโลก ใช้บริการจากองค์กรหลายพันแห่ง

วิธีการช่วยเหลือของเรา

การยืนยันตัวตนผู้ใช้ (eKYC) ▶

ยืนยันเอกสารตัวตนด้วย AI พร้อมการตรวจสอบความมีชีวิต (Liveness) และการป้องกัน Deepfake ยืนยันผู้ใช้ทั่วโลกภายในเวลาน้อยกว่า 20 วินาที ตามเกณฑ์ PDPA และกฎหมายท้องถิ่น

การยืนยันตัวตนธุรกิจ (KYB) ▶

ตรวจสอบทะเบียนอัตโนมัติจากระเบียน 600 ล้านฉบับ การระบุตัวตนผู้ถือหุ้นแท้จริง และการตรวจสอบเอกสารองค์กร ลดการยืนยันด้วยตนเองลง 75% ข้ามประเทศ 140 แห่งขึ้นไป

การคัดกรองการป้องกันการฟอกเงิน (AML) ▶

คัดกรองรายชื่อบุคคลถูกคว่ำบาตร (Sanctions) บุคคลที่มีสถานภาพทางการเมือง (PEP) และข่าวสารเชิงลบ (Adverse Media) จากแหล่งข้อมูลกว่า 11,000 แหล่ง ตรวจสอบอย่างต่อเนื่องเพื่อปฏิบัติตามข้อกำหนดของ ปปง. และมาตรฐาน AML ระหว่างประเทศ

การตรวจสอบธุรกรรม (Transaction Monitoring) ▶

การตรวจสอบธุรกรรมแบบเรียลไทม์ด้วยกฎที่สร้างไว้ล่วงหน้า 300+ รายการ และการตรวจจับความผิดปกติด้วย AI ลดเวลาการเตรียมการตรวจสอบลง 90% ด้วยการสร้างรายงาน FIU อัตโนมัติ

การป้องกันการฉ้อโกง (Fraud Prevention) ▶

ป้องกันการฉ้อโกงตลอดวงจร ประกอบด้วย การฉ้อโกงบัญชีใหม่ การฉ้อโกงตัวตน การยึดบัญชี การฉ้อโกงการชำระเงิน และการป้องกันการโอนเงินผิดกฎหมาย ตรวจสอบตัวตนสังเคราะห์และการฉ้อโกงเครือข่ายด้วยสัญญาณตัวตนเพิ่มเติม 1.4 พันล้านรายการ

KYC ที่นำมาใช้ซ้ำได้ (Reusable KYC) ▶

ผู้ใช้ยืนยันตัวตนเพียงครั้งเดียว แล้วเข้าถึงบริการได้ทั่วทั้งระบบนิเวศของพาร์ทเนอร์ อัตราผ่านการยืนยัน 92% อัตราสำเร็จเพิ่มขึ้น 30% และยืนยันตัวตนภายในไม่ถึง 10 วินาทีผ่าน API

การยืนยันตัวตนโดยไม่มีเอกสาร (Non-Document Verification) ▶

ยืนยันตัวตนโดยไม่ต้องใช้เอกสารผ่านฐานข้อมูลของรัฐ ธนาคาร และเครดิต เฉลี่ยการยืนยัน 4.5 วินาที ถึงลูกค้า 2.9 พันล้านรายข้ามประเทศ 20 แห่งขึ้นไป

การตรวจสอบความมีชีวิตและการป้องกัน Deepfake ▶

ความแม่นยำในการตรวจสอบ Deepfake 99.98% ในการพยายามครั้งแรก การวิเคราะห์พิกเซลด้วย AI ป้องกันหน้ากาก หุ่นขี้ผึ้ง AI อวตาร์ และบุคคลหน้าคล้าย ได้รับการรับรองจาก iBeta

การให้คำปรึกษาเรื่องการปฏิบัติตามกฎหมาย (Compliance Consulting) ▶

นำทางกฎหมาย PDPA ปปง. และ ธปท. ของไทยด้วยคำแนะนำจากผู้เชี่ยวชาญ บริการที่ปรึกษาด้านการปฏิบัติตามกฎหมายแบบครบวงจร ปรับให้เหมาะสมกับอุตสาหกรรมและตลาดของคุณ

การจัดการคดี (Case Management)

พื้นที่ทำงานที่เป็นศูนย์กลางสำหรับการแจ้งเตือน เอกสาร และการติดตามการตัดสินใจ

AI Copilot

เครื่องมือสร้างเวิร์กโฟลว์ (Workflow Builder)

เวิร์กโฟลว์การยืนยันแบบไม่ต้องเขียนโค้ด ด้วยตัวเรียกใช้ตามเงื่อนไขและการกำหนดเส้นทาง

การให้คะแนนความเสี่ยง (Risk Scoring)

การประเมินแบบเรียลไทม์ตามธุรกรรมและรูปแบบพฤติกรรมผู้ใช้

การวิเคราะห์แบบกำหนดเอง (Custom Analytics)

แดชบอร์ดและการรายงานการปฏิบัติตามกฎหมายที่ปรับแต่งตามความต้องการของคุณ

คำถามที่พบบ่อย

การยืนยันตัวตนลูกค้า (KYC) คือกระบวนการยืนยันตัวตนของลูกค้าของคุณ ในไทย ธุรกิจที่ได้รับการควบคุมตามข้อบังคับ ปปง. ธปท. หรือ ก.ล.ต. จะต้องดำเนินการ KYC แม้แต่ธุรกิจที่ไม่ได้รับการควบคุมก็ได้รับประโยชน์จาก KYC ในการป้องกันการฉ้อโกง ลดการเรียกคืนเงิน (Chargeback) และสร้างความไว้วางใจผู้ใช้
ลูกค้าส่วนใหญ่เปิดใช้งานได้ภายใน 1-3 วัน โดยใช้ Web SDK หรือ Mobile SDK ของเรา การเชื่อมต่อแบบ API มักใช้เวลา 1-2 สัปดาห์ ขึ้นอยู่กับข้อกำหนดทางเทคนิคของคุณ ทีมงานของเราพร้อมให้การสนับสนุนตลอดกระบวนการ
ใช่ All ID ปฏิบัติตามกฎหมาย พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล (PDPA) อย่างเต็มที่ ข้อมูลส่วนบุคคลทั้งหมดได้รับการประมวลผลตามข้อกำหนด PDPA โดยมีการลดข้อมูล การจัดการความยินยอม และและสิทธิในการลบข้อมูล ถูกรวมอยู่ในแพลตฟอร์มของเรา ข้อมูลสามารถประมวลผลภายในไทยหรือในภูมิภาคที่คุณเลือก
All ID รองรับการยืนยันตัวตนทั่ว 220+ ประเทศและอาณาเขต ด้วยประเภทเอกสารมากกว่า 14,000 รายการ - รวมถึงบัตรประชาชนไทย หนังสือเดินทาง ใบอนุญาตขับขี่ และใบอนุญาตทำงาน OCR รองรับสคริปต์หลักทั้งหมด รวมถึงไทย ซีริลลิก อาหรับ ฮินดี และจีน
Reusable KYC อนุญาตให้ผู้ใช้ที่ได้รับการยืนยันแล้วแบ่งปันการยืนยันของพวกเขาในบริการหลายบริการในอีโคซิสเต็มพาร์ทเนอร์ของคุณ แทนที่จะทำกระบวนการ KYC แบบเต็มซ้ำ ผู้ใช้เพียงแค่ทำการตรวจสอบความมีชีวิตอย่างรวดเร็วเพื่อยืนยันตัวตนของพวกเขา วิธีนี้ลดอุปสรรคในการใช้งาน เพิ่มอัตรา Conversion 30% และมีอัตราผ่านการยืนยัน 92%
All ID นำเสนอราคาแบบจ่ายตามการใช้งาน (Pay-per-verification) ไม่มีค่าธรรมเนียมล่วงหน้า คุณจ่ายเฉพาะเมื่อการยืนยันตัวตนสำเร็จเท่านั้น เรามีแพ็กเกจองค์กรปรับตามปริมาณและความต้องการเฉพาะของคุณ ติดต่อทีมงานเพื่อรับใบเสนอราคาเฉพาะบุคคล
